Spring Framework - Từ Sợ Hãi Đến Yêu Thích

6 tháng 10, 2025 Nguyễn Tiến Dũng

Lần đầu gặp Spring - Cú sốc văn hóa

Sau 4 tháng học Java core, tôi nghĩ mình đã "pro" rồi. Đến khi mentor bảo "Giờ học Spring đi", tôi Google và thấy hàng đống thuật ngữ: DI, IoC, AOP, Bean... Cảm giác như vừa học xong chữ cái lại phải học toán cao cấp!

Dependency Injection - Cái gì thế này?

Tuần đầu học DI, tôi hoàn toàn không hiểu tại sao phải phức tạp hóa code. Thay vì new UserService() đơn giản, lại phải dùng @Autowired. Mãi đến khi làm project lớn hơn mới thấy lợi ích khủng khiếp của việc này!

  • Trước Spring: 10 class, 50 dòng new Object(), debug cả ngày
  • Sau Spring: Chỉ cần annotation, Spring lo tất cả
  • Testing: Mock object dễ dàng, không còn nightmare

Spring Boot - Game changer thực sự

Khi chuyển từ Spring MVC thuần sang Spring Boot, tôi như được giải thoát! Không còn phải config XML dài ngoằng, không còn phải setup Tomcat thủ công. Chỉ cần @SpringBootApplication và chạy main method!

Những lần "đập đầu vào tường" với Spring

  • Circular Dependency: Mất 2 ngày để hiểu tại sao Bean không tạo được
  • @Transactional: Dữ liệu không save, hoá ra method không public
  • Component Scan: Bean không được tìm thấy vì sai package
  • Profile: Config dev khác prod, application crash liên tục

Hành trình từ newbie đến "biết xài"

Quá trình học Spring của tôi:

  • Tháng 1: Hoang mang với DI, chỉ copy-paste tutorial
  • Tháng 2: Hiểu @Component, @Service, @Repository
  • Tháng 3: Làm được REST API đơn giản
  • Tháng 4: Connect database, làm CRUD
  • Tháng 5: Security, JWT, role-based authorization
  • Tháng 6: Deploy lên cloud, cảm thấy "mình đã khá"

Những resource học Spring hay nhất

  • Spring.io Guides: Official tutorial, ngắn gọn súc tích
  • Baeldung: Giải thích chi tiết, ví dụ thực tế
  • YouTube - Java Brains: Giảng hay, dễ hiểu
  • GitHub projects: Clone về và chạy thử

Lời khuyên cho bạn sắp học Spring

  • Nắm vững Java core: OOP, Collection, Exception handling
  • Học từ từ: Core Spring trước, Spring Boot sau
  • Thực hành ngay: Mỗi concept học xong là code luôn
  • Đọc documentation: Spring docs viết rất tốt
  • Join cộng đồng: Spring VN Facebook group rất active

Kết luận

Spring khó ở chỗ concept, không phải syntax. Một khi hiểu được DI và IoC, mọi thứ sẽ sáng tỏ. Giờ tôi không thể tưởng tượng làm Java mà không có Spring. Nó đã thay đổi hoàn toàn cách tôi tư duy về architecture!
