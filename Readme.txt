# DũngWebBlog - Website Cá Nhân Nguyễn Tiến Dũng

## Giới thiệu
Website cá nhân DũngWebBlog được phát triển hoàn toàn bởi Nguyễn Tiến Dũng, trình bày thông tin chuyên môn và kinh nghiệm của một Full-stack Developer chuyên về Java và JavaScript.

## Thông tin Dự án
- **Tên dự án**: DũngWebBlog - Website Cá Nhân & Blog Chuyên Môn
- **Phiên bản**: 2.0.0
- **Ngôn ngữ**: Vietnamese (Tiếng Việt)
- **Tác giả**: Nguyễn Tiến Dũng
- **Công nghệ**: HTML5, CSS3, JavaScript ES6+, Bootstrap 5

## Cấu trúc Website

### 🏠 Trang chính (index.html)
- Header với navigation menu
- Hero section giới thiệu bản thân
- About section với thông tin cá nhân
- Skills showcase với các công nghệ thành thạo
- Contact form

### 👤 About (about.html)
- Thông tin chi tiết về bản thân
- **Skills & Technologies**:
  - **Java**: 85% - Spring Framework, Spring Boot, JPA/Hibernate
  - **JavaScript**: 80% - ES6+, DOM Manipulation, Async Programming
  - **Python**: 70% - Web Development, Data Processing
  - **HTML/CSS**: 90% - Responsive Design, Bootstrap, SCSS
  - **React**: 75% - Component-based Architecture, Hooks, State Management
  - **MySQL**: 78% - Database Design, Query Optimization, Joins, Stored Procedures

### 💼 Dự Án (portfolio.html)

- Chi tiết kỹ thuật và công nghệ sử dụng

### 📝 Blog System
Website bao gồm hệ thống blog hoàn chỉnh với **9 bài viết chuyên môn**:

#### Java Series:
1. **java-co-ban.html** - Java Cơ Bản: Biến, Kiểu Dữ Liệu và Cú Pháp
2. **java-oop.html** - Lập Trình Hướng Đối Tượng trong Java
3. **java-collections.html** - Java Collections Framework
4. **java-spring.html** - Spring Framework và Spring Boot

#### JavaScript Series:
5. **javascript-co-ban.html** - JavaScript Cơ Bản cho Người Mới Bắt Đầu
6. **javascript-es6.html** - JavaScript ES6+ và Các Tính Năng Mới
7. **javascript-dom.html** - DOM Manipulation với JavaScript
8. **javascript-async.html** - Asynchronous JavaScript và Promises

#### Tips & Tricks:
9. **java-javascript-tips.html** - Tips và Tricks cho Java và JavaScript Developers
